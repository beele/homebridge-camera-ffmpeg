You can also use "rtsp://ip-address/11" for HQ-Stream.

## config.json

```json
{
  "platform": "Camera-ffmpeg",
  "cameras": [
  {
    "name": "Camera",
    "videoConfig": {
    "source": "-rtsp_transport tcp -i rtsp://ip-address/12",
    "stillImageSource": "-i http://username:password@ip-address/tmpfs/auto.jpg",
    "maxStreams": 2,
    "maxWidth": 640,
    "maxHeight": 352,
    "maxFPS": 5,
    "vcodec": "h264_omx"
      }
  }
  ]
}
```
